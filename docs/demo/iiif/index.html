IIIF in image tags.

This is an example of overriding the behavior of an image tag.

Here is a standalone image tag where the url is to a json file. If the screen width is greater than 480px, it will render as a IIIF viewer using OpenSeaDragon.

The image src passed in the markdown is https://iiif.archivelab.org/iiif/byte-magazine-1982-08$295/info.json
'

A regular image will behave normally.
